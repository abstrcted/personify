# Credits & Acknowledgments

## APIs & Services

### GetSongBPM API
This project uses the [GetSongBPM API](https://getsongbpm.com/api) for retrieving detailed audio features including BPM, key, energy, danceability, and other track characteristics.

**GetSongBPM** provides comprehensive music data that helps power our personality-music matching algorithm.

### Spotify Web API
We use the [Spotify Web API](https://developer.spotify.com/documentation/web-api) for user authentication and accessing user music libraries and listening history.
